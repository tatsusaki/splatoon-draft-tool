# ドラフト支援ツール 仕様書（ラフ）

※最初はざっくりで構いません。後で少しずつ更新していきます。

## ツールの目的

- スプラトゥーンのドラフト形式大会におけるドラフト進行を支援する
- 手作業では面倒な管理を、スプレッドシート + GAS で自動化して運営負荷を下げたい

## 主な機能（箇条書き）

- 参加者一覧管理
- フォームを使った選手指名
- 指名被り時の処理（優先Pt→ルーレット）
- 各キャプテンの有線Pt管理
- 選手、観戦者へのドラフト状況を示すインターフェース
- ルーレットなどのUI表示

## 非機能要件（ざっくり）

- 試合のテンポを損なわない程度の応答時間（理想：ボタン押下後 1〜2 秒程度）
- GAS の実行時間制限に引っかからないこと
- 新しいルール・ロジックを追加しやすい構造であること
